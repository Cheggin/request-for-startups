# social-media-agent

**Status:** 🔴 Not started
**Agent:** content
**Category:** content
**Created:** 2026-04-13

## Description

Content agent creates social media posts (Twitter/X, LinkedIn) about the startup. Uses SOUL.md for voice and brand consistency. Schedules posts on a cadence. Tracks engagement metrics. Reports social performance in investor updates.

## Checklist

- [ ] Twitter/X API integration for posting and reading engagement
- [ ] LinkedIn API integration for posting and reading engagement
- [ ] Post generation using SOUL.md voice/tone guidelines
- [ ] Post types: product announcements, tips, behind-the-scenes, user stories
- [ ] Scheduling system — consistent posting cadence (e.g., 3x/week per platform)
- [ ] Engagement tracking — likes, reposts, replies, impressions per post
- [ ] Best-performing post formats identified and repeated
- [ ] Social metrics summarized in investor updates
- [ ] Brand guidelines (from brand-guidelines feature) enforced in all posts
- [ ] No duplicate content across platforms — adapted per platform norms
- [ ] Thread support for Twitter/X for longer-form content

## Notes

- SOUL.md defines the startup's voice — the content agent must read it before every post
- LinkedIn tone is typically more professional; Twitter/X is more casual — adapt accordingly
- Engagement data feeds into the growth agent's overall marketing performance view
